DOKUMENTY PDF
=============

Ten folder jest przeznaczony na pliki PDF dostępne do pobrania na stronie /dokumenty.

Jak dodać plik:
1. Umieść plik PDF w tym folderze (np. regulamin.pdf)
2. Zaktualizuj ścieżkę fileUrl w /src/data/documents.ts
   Przykład: fileUrl: '/documents/regulamin.pdf'

Oczekiwane pliki (do zastąpienia):
- regulamin.pdf        → Regulamin zajęć
- rodo.pdf             → Zgoda RODO
- umowa.pdf            → Umowa z rodzicami
- program.pdf          → Program zajęć
- cennik-2024-2025.pdf → Cennik 2024/2025

UWAGA: Plik placeholder.pdf nie istnieje — zastąp go prawdziwymi plikami.
